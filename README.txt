== Description ==

This plugin provides pricing tiers for WooCommerce. 

*Note* Currently only works with simple products.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `easy-wc-tiers` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== How To Use ==

1. Create or edit a product in WooComerce. 
2. In the 'Product Data' box, select the 'Pricing Tiers' tab.
3. Add a tier, provide a minimum and maximum quantity, and the discount amount (e.g. 5). This will be either 5% or $5 off each product.
